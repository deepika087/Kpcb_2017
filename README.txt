Prerequisite
	Java 7/8

1. Import the project in IDE of your choice 
2. Either use Run from eclipse/Netbeans/IntelliJ on CustomHashMap.java
Or
1. Go to the root directory of this project
2. Go to Terminal and traverse to root directory of this project
   Say, cd [YourPath]/Deepika_KPCB
   Such that using ls here should show the following content
   README.txt	bin		src
3. cd src/
4. Make sure you have java in the path varible for the next step
5. javac CustomHashMap.java 
   The above will generate CustomHashMap.class and Entry.class
6. java CustomHashMap
